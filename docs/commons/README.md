# Taegis SDK for Python

The Taegis SDK is a Python library for interfacing with the GraphQL APIs in Taegis.  The following are common API implementations to reduce API call setup and pagination.  Commons projects are also meant to quick interface with each other.  Function and methods input and outputs should align to chain results.
